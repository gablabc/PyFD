==================

.. image:: Images/Header_PyFD.png
   :width: 1000px
   :align: center

**PyFD** (Functional Decomposition in Python) is a package for decomposing complex black-box models into simpler components.

Install
=======
The package must be installed manually 

.. code-block:: console

   git clone https://github.com/gablabc/PyFD.git
   cd PyFD
   python3 -m pip install .
